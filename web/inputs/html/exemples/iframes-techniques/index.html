Masquer les iframe techniques de manière accessible

Introduction

On entends par iframe technique une iframe qui a un rôle purement technique mais n'est pas visible aux utilisateurs, par exemple une iframe qui execute un script pour compter le nombre de visiteurs d'un site.

Hors, les référentiels d'accessibilité nous demandent d'indiquer dans l'attribut title le contenu qui sera présent dans l'iframe, information qui sera accessible aux outils d'assistance (un lecteur d'écran vocalisera l'attribut title d'une iframe).

Cacher l'iframe à tous les utilisateurs

La solution pour cacher l'iframe y compris aux utilisateurs d'outils d'assistance est de mettre un attribut hidden sur l'iframe. Les scripts contenus dans l'iframe seront executés.

Il est toujours obligatoire de renseigner l'attribut title de l'iframe.


      <iframe title="contenu technique" hidden src="http://www.orange.com/"></iframe>
